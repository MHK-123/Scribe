Welcome to the **Technical Rituals Manifest**, your guide to commanding the Scribe sentinel.

## 🛡️ Command Security
> [!IMPORTANT]
> **Channel Restriction**: Prefix commands (`$l`, `$m`, etc.) are limited to the **single channel** configured in your Web Dashboard. 
> - Usage in other channels will trigger an automated deletion and a DM notification.
> - This ensures zero clutter and maintains sanctuary purity.

---

## 🔊 Voice Sanctuary Commands (Slash Commands)
*These commands empower party leaders to manage their dungeons in real-time.*

| Command | Permission | Description |
|---------|------------|-------------|
| `/vc-name [name]` | Owner | Renames your active voice sanctuary. |
| `/vc-status [text]`| Anyone | Sets a custom status for the voice channel. |
| `/vc-kick [user]` | Owner | Banishes a hunter from your current party. |
| `/vc-ban [user]` | Owner | Permanently bars a user from entering your sanctuary. |
| `/vc-unban [user]` | Anyone | Revokes a ban for a specific hunter. |
| `/vc-transfer [user]`| Anyone | Pass the leadership (Owner) of the sanctuary. |
| `/vc-lock` | Anyone | Seals the channel (Private Ritual). |
| `/vc-unlock`| Anyone | Unseals the channel for all hunters. |
| `/vc-invite [user]`| Anyone | Summons a specific hunter (bypasses locks). |
| `/pomodoro start` | Anyone | Manifests the focus engine in the current VC. |

## 🤖 Automated Sanctuaries
Scribe operates on high-fidelity automation cycles:

1. **Auto-Manifestation**: Joining a "Join-to-Create" channel spawns a personalized sanctuary instantly.
2. **Auto-Purge**: Sanctuaries are deconstructed the moment the last hunter leaves.
3. **Residency Tracking**: XP is earned globally across all voice realms while party residency (2+ users) is maintained.
4. **Pomodoro Disposal**: HUD messages are deleted immediately upon channel vacancy to prevent ghost manifestations.
