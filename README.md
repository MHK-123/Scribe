**Scribe** is a premium, full-stack Discord ecosystem designed to transform study sessions into high-stakes dungeon raids. Built with a sleek, dark-mode aesthetic and hardened security, it seamlessly integrates a **Discord Bot**, a **Real-time Backend**, and an **Interactive Web Dashboard**.

---

## ⌛ THE SCRIBE ENGINE
The heart of the Sanctum. Scribe automates your productivity rituals:
- **Scribe HUD**: Joining a configured Voice Channel instantly manifests a real-time Pomodoro HUD, tracking your session focus with cinematic precision.
- **Temporal Calibration**: Global timezone-aware resets ensure rewards and leaderboards align with your server's local midnight.
- **Hunter Ranking**: Earn XP for every second spent in focus. Level up your Hunter rank and unlock exclusive realm rewards.

---

## 🌌 ARCHITECTURE & STACK
- **The Vessel (Frontend)**: React + Vite + Tailwind (Premium Dark Theme).
- **The Core (Backend)**: Node.js + Express + Socket.IO (Real-time synchronization).
- **The Ledger (Database)**: PostgreSQL (Reliable state persistence).
- **The Sentinel (Bot)**: Discord.py (Advanced interactions & HUD management).

## 📜 QUICK START
Ready to initialize the engine? Follow the [Installation Manifest](guide.md) for a step-by-step setup guide.

### Basic Initialization:
1. **Calibrate**: Fill in your `.env` files across all directories.
2. **Ignite**: Run the Backend, Frontend, and Bot services.
3. **Sync**: Use `.sync` in your Discord server to propagate the Scribe nodes.
